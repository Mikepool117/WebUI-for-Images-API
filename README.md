
# A WebUI For The Images API

A WebUI made using streamlit for the OpenAI Images API which uses the DALLÂ·E model.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Mikepool117/WebUI-for-Images-API
```

Create a Virtual Environment

```bash
  python -m venv WebUI-for-Images-API
```

Go to the project directory

```bash
  cd WebUI-for-Images-API
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run the app

```bash
  streamlit run main.py
```


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
